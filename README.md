# BountyTriage
A lightweight, browser-based utility for security researchers and SOC analysts to quickly categorize and prioritize vulnerability reports.

## Features
- **Zero Dependencies**: Pure HTML/CSS/JS.
- **Fast Triage**: Instant priority calculation based on impact.
- **Mobile Friendly**: Works on any device.

## Usage
Simply open `index.html` in any modern web browser or visit the live GitHub Pages link.

## Project Goal
This project was built to streamline the initial assessment phase of bug bounty hunting and vulnerability management.
